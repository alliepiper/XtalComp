XtalComp: Compare crystal structures

XtalComp Web Interface: Enter POSCAR files

Paste a POSCAR file into each of the following boxes and hit "Compare" to let XtalComp determine if they describe the same structure.

Notes:

  • The POSCAR format is specified here.
  • Each structure must have the same number of atoms, and the order of the atom type counts (line 6) must be the same in each POSCAR.
  • Update: you may now compare different crystals with different formula units (e. g. Ti1O2 and Ti2O4). The crystals will be reduced to their primitive form using functions from spglib before the comparison. However, the order of atoms must still match.

About:

XtalComp is an algorithm developed by David C. Lonie for the genetic algorithm XtalOpt. It will determine if two crystal structure descriptions refer to the same underlying structure, despite choice of unit cell vectors, rotations, reflections, translations, and numerical noise. The source code is available under the "New" BSD open source license from our Github repository. The algorithm has been fully described in an academic journal (free preprint).
